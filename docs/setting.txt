로컬 실행 시에는 .env 파일이, 배포 시에는 .env.production 파일이 사용되도록 설정하는 것이 일반적입니다.
수정해야 할 파일과 변경사항
package.json 파일 수정
"build": "nuxt build",
-->
"build": "cross-env NODE_ENV=production nuxt build",
이렇게 수정하면:
npm run dev (로컬 실행) → .env 파일 사용
npm run build (배포 빌드) → .env.production 파일 사용
cross-env는 이미 프로젝트에 설치되어 있으므로 추가 설치 없이 바로 사용 가능합니다.
